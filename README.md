# SOEN357: Project
Project for the SOEN 357 course at Concordia University Canada for the Winter 2025 semester

## Description
üåç WorldEats is a mobile application designed to help users confidently explore and cook culturally diverse meals while accommodating their dietary needs and lifestyles. The app features guided recipes, cultural context cards, dietary filters, meal planning tools, and a community space for sharing tips and experiences. Built with a user-centered design approach, WorldEats aims to promote healthy eating habits, cultural appreciation, and an engaging cooking experience through personalized, intuitive interactions.

## How to Run the Project:
1. Clone the Repository:
```
git clone https://github.com/Sawanoza/SOEN357.git
```

2. Navigate to the application folder:
```
cd WorldEats
```

3. Install Dependencies
```
npm install
```

4. Start the Development Server
```
npx expo start
```
